## How to run this projects

##### Windows:

Step - 1: To Install Bun Your Local Machine
```bash
npm install -g bun
```

Step - 2:
```bash
git clone https://github.com/iamalshahed/love-calculator
```

Step - 3:
```bash
cd .\love-calculator\
```

Step - 4:
```bash
bun install
```

Step - 5:
```bash
bun run dev
```
